CSS Box Model Demonstration

Content-Box Model

Content-Box Model Content: The width and height only include the content. Padding, border, and margin are added to the total size of the element.

This box uses the box-sizing: content-box; model. The total width and height of the box are calculated as:

  • Width = Content Width + Padding + Border + Margin
  • Height = Content Height + Padding + Border + Margin

Border-Box Model

Border-Box Model Content: The width and height include the content, padding, and border. Margin is added outside the box.

This box uses the box-sizing: border-box; model. The total width and height of the box are calculated as:

  • Width = Content Width (including Padding and Border) + Margin
  • Height = Content Height (including Padding and Border) + Margin
